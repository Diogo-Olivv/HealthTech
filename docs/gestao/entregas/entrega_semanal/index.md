## Como preencher

1. Copie `entrega_template.md` para um novo arquivo `semana-NN.md`.
2. Preencha todas as seções com informações reais da semana.
3. Remova seções sem atividade.
4. Abra PR na branch `docs` com a mensagem:
   `docs(entrega): adiciona entrega da semana NN`.
